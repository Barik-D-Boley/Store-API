Using node to get data from the API

This example requires node and nodemon to be downloaded to the project. Type npm i node and npm i nodemon into your IDE's terminal to download node and nodemon.

Get all Shoes

To get every shoe in the api, go to the url and replace

/index.html

with

/api/v1/products

Get a Specific Shoe

To get a single shoe in the api, go to the url and replace

/index.html

with

/api/v1/products/:id

. ":id" is the specific id from the shoe, which can be found in the products list.

The response should look similar to this:

{
----"product": {
--------"_id": "61bbb0f62e02dcfeb9a60325",
--------"name": "SB Zoom Blazer Mid Premium",
--------"price": 105,
--------"featured": true,
--------"rating": 41,
--------"createdAt": "2021-12-16T21:34:45.445Z",
--------"company": "Nike",
--------"__v": 0
----}
}

Add a Shoe to the API

The page needs information on it for the API to be able to post, so go to the Product Page to post a new shoe.

The response should look similar to this:

{
----"product": {
--------"name": "*name*",
--------"price": *price*,
--------"featured": false,
--------"rating": *rating*,
--------"createdAt": "*Date Created*",
--------"company": "Nike",
--------"_id": "*id*",
--------"__v": 0
----}
}
